## Requirements
#### [Java 8+](https://www.oracle.com/java/technologies/downloads/)
## How to Run
1. **Compile code**
- run command 
```console
javac lns/tasks/*java 
```
- make sure you are in correct directory
```console
your\path\RecruitmentTasks\src\main\java
```
2. **Run application**
```command
java lns.tasks.Main
```
- make sure you are in correct directory
```console
your\path\RecruitmentTasks\src\main\java
```
